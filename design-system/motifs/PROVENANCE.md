# Motif Library Provenance

All SVGs in this folder are **original works** drawn for Te Kete Toi on documented, generalised (broader-zone) structures — no iwi/hapū-attributed variants, no reproduction of any artist's work. They use `currentColor` so context sets the colour.

| File | Name | Documented meaning (see docs/pattern-glossary.md) | Zone |
|---|---|---|---|
| koru.svg | Koru | growth, new life, the cycle of life | broader |
| wai.svg | Wai | generalised water rhythm (deliberately NOT a carved pūhoro rendering) | broader |
| poutama.svg | Poutama | stepped ascent toward knowledge; whakapapa | broader |
| purapura-whetu.svg | Purapura whetū | star seeds — the multitudes | broader |
| kaokao.svg | Kaokao | chevron — strength, the braced sides of warriors | broader |
| tuna.svg | Tuna line | original motif for Te Ara Tuna (author: Brett Gillett kaupapa) | author's own |

Source: Te Kete Toi (original, MIT). Attribution: none required beyond repo credit.
Rule: additions to this folder require the same table entry plus significance-layer review. Iwi-attributed or commissioned motifs go in `assets/patterns/` with full provenance sidecars instead.
