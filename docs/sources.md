# Annotated Sources — Ngā Puna Mātauranga

Freely accessible sources this knowledge base cites. Verify against originals before relying on any claim; correct with knowledge-holder input where offered.

## Encyclopaedic / institutional

- **Te Ara — Encyclopedia of New Zealand** (teara.govt.nz) — Whakairo (Māori carving) entries; 1966 Encyclopaedia Māori art pages (surface patterns, painted designs, culture areas). Core citable baseline for pattern names and regional styles.
- **Te Papa Tongarewa** (collections.tepapa.govt.nz) — object records with iwi attributions; many images openly licensed. Check per-object licences.
- **DigitalNZ** (digitalnz.org) — aggregated search across NZ collections with licence filters.
- **Christchurch City Libraries — Ngā Kete Wānanga o Ōtautahi** — Toi Māori and tukutuku panel guides with pattern meanings.
- **Whanganui Regional Museum notebooks** — regional carving and weaving āhuatanga.
- **Science Learning Hub — Pokapū Akoranga Pūtaiao** (sciencelearn.org.nz) — whakairo with mātauranga Māori framing.

## Educational

- **ANZAAE** (anzaae.nz) — kōwhaiwhai and tukutuku classroom guides (e.g. "Kōwhaiwhai for beginners").
- **NZQA unit standards** (e.g. 18694 tukutuku, 2598 whakairo surfaces) — codified practice descriptions.
- **The New Code — "World Ethnomathematics: Maori Designs in SVG"** — worked SVG constructions of kōwhaiwhai symmetry.

## Tikanga / governance

- **Waitangi Tribunal, Wai 262 — Ko Aotearoa Tēnei** — kaitiaki interests in taonga works.
- **Karaitiana Taiuru — Kaupapa Māori AI Framework** (taiuru.co.nz) — including the atarangi/shadow principle this repo adopts.
- **Te Mana Raraunga — Māori Data Sovereignty principles**.
- **"The five tests: designing and evaluating AI according to indigenous Māori principles"** — AI & Society (Springer).
- **"Kaupapa Māori concept modelling for the creation of Māori IT artefacts"** — Journal of the Royal Society of NZ.

## Pattern asset sources (check licences per item)

- Te Papa openly-licensed collection images; DigitalNZ licence-filtered results; The Noun Project "Māori" icon sets (attribution licences); commissioned work via `docs/designers.md` — always with provenance sidecars.

## Not acceptable as sources

Stock-art sites' "Māori-style" vectors without provenance; AI-generated pattern imagery; Pinterest-grade uncited collections; any source that can't be traced to a person, institution, or iwi.
