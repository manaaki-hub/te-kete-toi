# Pattern Assets — Provenance Required

Only provenanced assets live here. Every `name.svg` needs `name.provenance.yml` (schema in TIKANGA.md §5). Assets without provenance are invisible to all agents.

Acceptable sources: commissioned work (via docs/designers.md), whānau/hapū-supplied designs within their permission scope, openly licensed institutional images (Te Papa/DigitalNZ — check per-object licence), your own original work built on documented generic structures.

Not acceptable: AI-generated "Māori-style" imagery, unprovenanced stock vectors, scraped artwork.

Folder convention:
```
assets/patterns/
  kowhaiwhai/   tukutuku/   whakairo/   taniko/   contemporary/
```
